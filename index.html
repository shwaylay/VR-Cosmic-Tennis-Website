VR Cosmic Tennis

Michael Suehle & Daniel Lopez

CMSC838C Final Project

Overview

VR Cosmic Tennis is a game where a player in VR flies around in a jetpack and tries to hit tennis balls back toward their opponent.
Checkout the source code on our Github Repo!

Goals

  • Implement tilt-based jetpack controls
  • Implment tennis ball hitting physics
  • Create an alien creature opponent
  • Add HRTF sound effects
  • Add gravitational obstacles
  • Create an interesting space-themed environment
